# 📊 Acadex – Student Performance Dashboard

## 📌 About the Project

Acadex is a modern and interactive Student Performance Dashboard designed to analyze and visualize students' academic performance. Built using **HTML, CSS, JavaScript, Chart.js, and JSON**, the application provides administrators and teachers with an intuitive interface to monitor student progress, compare subject-wise performance, generate report cards, and view detailed analytics through interactive charts. The dashboard features a responsive design, real-time search, theme customization, and data-driven insights to simplify academic performance management.

## 🎯 Project Objective

The objective of Acadex is to provide an efficient and user-friendly dashboard for analyzing student performance. It helps educators visualize academic progress, identify strengths and weaknesses, monitor overall class performance, and generate detailed student reports through an interactive and responsive interface.
